# Cashier Interface Guide

## Module 1: Dashboard

### Dashboard
The Dashboard Screen displays your sales overview including total sales, returns, transaction counts, and daily performance metrics to help you monitor your cashier operations in the SOFTWEAR platform.

---

## Module 2: Transaction Processing

### Sales
The Sales Screen lets you process customer sales transactions using the point-of-sale system, add products to cart, and complete purchases in the SOFTWEAR platform.

### Returns
The Returns Screen lets you process product returns and refunds for customer transactions, including pending return approvals in the SOFTWEAR platform.

---

## Module 3: Reports

### Sales Report
The Sales Report Screen displays detailed reports of all sales transactions you've processed with filtering and search options in the SOFTWEAR platform.

### Returns Report
The Returns Report Screen displays detailed reports of all return and refund transactions you've processed with filtering options in the SOFTWEAR platform.

---

## Module 4: Settings & Logs

### Settings
The Settings Screen lets you update your personal information, address, email, password, and account preferences in the SOFTWEAR platform.

### Activity Log
The Activity Log Screen lets you view your personal activity history including all sales and return transactions you've performed in the SOFTWEAR platform.
